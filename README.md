## 初見まとめ

分かんなくて調べたこと。

### html

```html
<meta name="description" content=" " />
```

サイトの概要文、検索をしページをクリックする前に書かれてある文章。

### css

```css
text-decoration: none;
```

テキストの装飾、none は初期値。

```css
height: calc(100vh - 60vh);
```

calculation（計算）100vh から 60vh 引いた値（40vh）で表示。vh はフルスクリーン時の割合値。
